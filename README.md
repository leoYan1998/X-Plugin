# 🛠️ X (Twitter) 批量取消非回关脚本

这是一个基于 **Tampermonkey（油猴）** 开发的个人高定制脚本。针对 X (Twitter) 极其严格的风控机制以及复杂的动态页面架构进行了闭环优化。具备“双速度滑块调整”、“安全数量熔断阀门”和“黑客绿实时日志流监控”，支持半自动与后台全自动清理，真正做到不占用物理鼠标。

---

## ✨ 核心功能

* **双模式一键切换**：
* **半自动**：用鼠标手动滚动页面，脚本在视野内自动捕捉并秒杀非回关用户。
* **全自动**：脚本全权接管滚屏与判定。支持丢在后台标签页或最小化挂机，不抢占鼠标焦點。


* **双参数实时微调**：
* **取关间隔上限**：1秒至8秒动态可调，模拟真人的随机风控等待期。
* **滚屏刷新速度**：0ms至2000ms可调，支持 **`⚡极限极速 (0ms)`**，突破切碎等待，解除物理刹车片。


* **🎯 数量安全阀门**：
* 提供**每日清理上限输入框**（默认 80 人）。一触及设定数字，脚本瞬间抛出异常进行硬熔断，锁死后续一切点击与滚动，彻底杜绝挂机睡过头导致被官方封号的风险。


* **📋 黑客绿实时日志区**：
* 面板底部集成独立日志流（LogBox），成功取关后实时输出账户 ID（如 `[1] ✅ 已取关 @username`），支持日志区自动置底滚动，运行状态一手掌握，不再依赖弹窗开盲盒。


* **防重记忆库 (Set)**：
* 自动记忆已被判定或已取关的账号，完美克服 Twitter 前端 React 动态虚拟列表（Virtual List）重复加载导致的漏人或卡死问题。



---

## 🚀 安装与使用

### 1. 安装脚本

1. 确保浏览器已激活 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. 点击油猴图标，选择“添加新脚本”。
3. 清空默认模板，将 `script.js` 中的完整代码复制进去并保存（`Ctrl + S` 或 `Cmd + S`）。

### 2. 使用步骤

1. 访问你的 **X 个人主页** 的正在关注列表：
`[https://x.com/你的用户名/following](https://x.com/你的用户名/following)`
2. 页面加载完成后 2 秒，网页右上方会自动浮现半透明的黑色控制面板。
3. **设定安全防线**：在 `🎯 本次清理上限` 输入框中输入你预期的数量（建议保持默认的 80）。
4. **调节速度滑块**：根据你当下的网络质量，将滚屏速度和间隔拉到适合的区间。
5. **一键启动**：
* 点击 **`▶️ 半自动守株待兔`**：手动滚轮，脚本代点。
* 点击 **`🚀 开启全自动清理`**：脚本全自跑，你可以去刷微信、打游戏或写代码。


6. **熔断与收尾**：中途可随时点击 **`🛑 停止清理`** 进行安全收尾；若触及上限阀门，脚本会自动刹车并弹出结算通知。

## ⚠️ 极其重要的安全与防封告示

虽然本脚本已经过高频调优并提供了多层防线，但 **X (Twitter) 官方对账号单日的取消关注（Unfollow）总量有着极强的隐性阈值监控**：

1. **单日限量**：极度建议每天流水线式清理 **不超过 100 个** 账户。
2. **细水长流**：如果非回关存量较大（如几千个），请分天、分批次挂机运行，切勿一次性拉满上限。
3. **极速代价**：开启 **`⚡极限极速`** 配合最激进的延迟虽能实现秒级取关，但高频动作极易在几分钟内触发官方频控。这可能导致账号被锁小黑屋（24 小时内禁止关注/取关）或触发高难度人机验证。请务必根据自身账号权重谨慎飙车。
