# X (Twitter) 批量取消未回关脚本

基于 **Tampermonkey（油猴）** 开发，用于精准识别并清理未回关的 X 账号。支持全自动、半自动以及“手动核对”三种流派的操作，并提供速度调节、数量防封锁熔断和实时日志面板。

---

## 功能

- **两种核心运行模式**
  - **全自动模式**：脚本自动滚动页面并检测/取消未回关账号，支持挂机后台运行。
  - **半自动模式**：脚本不自动滚屏，仅监控当前可见视窗。需用户手动滚动页面，脚本自动处理新刷出的账号。
- **🌟 手动核对确认流（自由开关）**
  - **开启（默认）**：检测到未回关直接自动点击取关。
  - **关闭**：检测到未回关时，脚本会**将该用户移至屏幕中央并高亮框选，同时自动挂起暂停**。由您人工决定是否点击官方取关。处理完后可一键点击“继续”或“结束并结算”。
- **灵活的参数调节**
  - 取消关注间隔上限：**1～8 秒**（随机延迟，模拟真人）。
  - 滚屏刷新速度：**0～2000 ms**（0ms 为极限速度，若网速慢易误判触底）。
- **安全熔断阀门**
  - 可自定义单次清理上限（默认 **80** 人），达到上限后脚本“自杀式”强行熔断停止，防止触发平台风控。
- **当前视窗防重**
  - 自动跳过当前可见列表中已扫描过或正在处理的账号，确保单次循环内不重复触发。
- **实时日志区**
  - 独立面板实时打印处理进度、锁定名单以及系统状态。

## 安装

1. 浏览器安装 **Tampermonkey（油猴）** 插件。
2. 点击油猴图标 ➔ “添加新脚本”。
3. 将最新的脚本源码完整复制进去并保存。

---

## 使用

1. 在浏览器中打开你的关注列表页面：
   `https://x.com/你的用户名/following` 或者 `https://twitter.com/你的用户名/following`
2. 等待 2 秒左右，右侧会自动弹出悬浮控制面板。
3. 根据需求调整参数：
   - 本次清理上限
   - 取消关注间隔上限
   - 滚屏刷新速度
   - **是否开启“发现目标自动取关”**
4. 点击对应的启动按钮：
   - **半自动** / **全自动**
5. **在“关闭自动取关”的人工核对场景下**：
   - 抓到目标后脚本挂起，此时可点击 **`继续`** 让脚本接着往下走；
   - 若觉得清理得差不多了，可点击橙色的 **`🏁 结束并结算`** 按钮，脚本会安全收网，弹出最终的战果统计弹窗。
6. 自动模式下，也可随时点击 `🛑 停止清理` 进行安全收尾。

---

## 参数说明

| 参数 | 默认/范围 | 说明 |
| :--- | :--- | :--- |
| **取消关注间隔上限** | 1～8 秒 | 每次自动取关后的安全等待时间最大值（随机波动） |
| **滚屏刷新速度** | 400ms (0～2000 ms) | 当前屏幕扫描完后，向下滚屏的停顿间隔 |
| **本次清理上限** | 80人 (自定义) | 强行防封熔断器，达到此累计数量后脚本彻底自杀停止 |
| **发现目标自动取关** | ☑️ 开启 (Checkbox) | 开启则全自动清人；关闭则遇敌暂停、高亮锁定，等待人工处理 |

---

## 注意事项

- **安全红线**：强烈建议每日取消关注不超过 **100** 个账号。
- 大量账号请务必分批、分多天清理。
- 将“取消关注间隔”拉得过低或将“滚屏刷新速度”设为极限速度，会极大增加平台触发人机验证（Captcha）或封禁 API 的风险。
