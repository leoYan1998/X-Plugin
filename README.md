# 海王管理大师（𝕏 批量取消非回关、自动关注未上线）

专为 𝕏 (Twitter) 打造的批量清理非回关账户的 Tampermonkey 脚本。

## 核心功能

- **智能识别**：精准识别双向关注，绝不误伤好友
- **自定义控制**：取关间隔、滚屏速度、清理上限可调
- **双模式**：自动取关 / 手动确认
- **白名单**：保护重要账户，支持快捷操作
- **安全机制**：随机延迟、熔断保护、防人机验证

## 安装使用

1. 安装 Tampermonkey 插件
2. 新建脚本，复制 `script.js` 代码保存
3. 访问 `https://x.com/[你的用户名]/following`
4. 配置参数后点击 **🚀 启动程序**

## 安全建议

- 每日取关不超过100人
- 间隔设置3-5秒
- 首次使用建议手动模式
