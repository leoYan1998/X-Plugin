# 🛠️ X (Twitter) 批量取消非回关脚本 (极致体验版)

这是一个基于 **Tampermonkey（油猴）** 开发的浏览器脚本，专为 X (Twitter) 用户设计。它可以自动识别并批量取消关注那些**没有回关你**的账户，支持半自动、全自动双模式，并配备了实时速度调节面板和安全防封机制。

---

## ✨ 核心功能

* **双模式一键切换**：
* **半自动（守株待兔）**：由你手动滚动页面，脚本在屏幕中自动捕捉并清理非回关用户。
* **全自动（彻底挂机）**：脚本自动滚屏、自动捕捉、到底自动停止，不占用物理鼠标，可切到后台运行。


* **实时双参数调节**：
* **取关间隔上限**：调节点击取关后的安全等待时间。
* **滚屏刷新速度**：根据网络状况，实时调节全自动流下的滚屏速度，支持 **0ms 极限极速模式**。


* **去重记忆库 (Set)**：针对 X 的“虚拟列表”技术深度优化，已扫描用户绝不重复检测，绝不漏人。
* **毫秒级强行中断**：配备异步熔断机制，按下“停止清理”后可在 0.1 秒内响应并安全刹车。
* **清单结算报告**：无论以何种方式结束，均会弹出系统级 `alert` 清单，详细列出本次操作成功取关的所有用户 ID，且控制台同步打印。

---

## 🚀 安装与使用

### 1. 安装脚本

1. 确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. 打开油猴面板，选择“添加新脚本”。
3. 将 `script.js` 中的完整代码复制进去并保存（`Ctrl + S`）。

### 2. 使用步骤

1. 打开浏览器，进入你的 **X 个人主页** 的正在关注列表，或直接访问：
`https://x.com/你的用户名/following`
2. 页面加载完毕 2 秒后，网页右上方会出现黑色的**控制面板**。
3. **调节参数**（可选）：根据你的网络与需求，拖动滑块调整速度。
4. **选择模式启动**：
* 点击 **`▶️ 半自动守株待兔`**：用鼠标往下滚屏，释放双手点击。
* 点击 **`🚀 开启全自动清理`**：脚本接管滚屏，你可以将标签页切到后台做别的事。


5. **停止与结算**：需要结束时点击 **`🛑 停止清理`**，浏览器将弹出本次取关的完整名单。

## 🎮 面板参数控制指南

| 功能组件 | 范围 | 说明 | 推荐设置 |
| --- | --- | --- | --- |
| **⏳ 取关间隔上限** | 1秒 - 8秒 | 取关单个人员后，在安全期内随机等待的时间上限。 | **安全流：4-5秒**<br>

<br>**激进流：1-2秒** |
| **🚀 滚屏刷新速度** | 0ms - 2000ms | 全自动模式下，屏幕无目标时向下滚屏的停顿间隔。 | **网络卡顿：1200ms+**<br>

> 📌 **注（5次停顿现象）**：在极速模式下，网页每取关 5 次左右可能会有短暂的物理停顿。此为 X 网页底层 React 架构进行 DOM 树重绘与垃圾回收导致的浏览器正常硬件延迟，非脚本卡死。

## ⚠️ 极其重要的安全与防封告示

虽然脚本内置了伪装真人的随机延迟机制，但 **X (Twitter) 官方对账户每天的取消关注总量有绝对的硬性封顶限制**（通常根据账号权重不同，每日限制在 100-200 人左右）。

1. **单日限量**：建议每天流水线式清理 **不超过 100 个** 账户。
2. **细水长流**：如果你有几千个非回关需要清理，请分天、分批次运行。点击 `🛑 停止清理` 结算后，明天再继续。
3. **超速代价**：将滑块拉满至 **`⚡极限极速`** 虽爽，但极易在极短时间内触发 Twitter 官方的单日频控，可能导致账号被关小黑屋（24 小时内禁止关注/取关）或触发人机验证。请谨慎飙车！
