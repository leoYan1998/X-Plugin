# 𝕏 批量取消非回关 (海王管理大师)

一款专为 𝕏 (Twitter) 打造的轻量级、智能批量清理非回关（单向关注）账户的 Tampermonkey 脚本。采用高仿真人操作逻辑，配备可视化控制面板与实时日志，助你高效、安全地管理关注列表。

---

##  核心功能

* **智能回关识别**：精准识别 `Follows you` / `关注了你` 标签，绝不误伤双向关注的好友。
* **双模式自由切换**：
    * **⚡ 自动取关模式**：开启后，雷达发现目标自动模拟点击并确认取关，全程无需介入。
    * **🔍 人工锁定模式**：关闭自动取关时，发现目标将**自动暂停滚动并高亮锁定账户**，由你手动决定留存或取关，按“继续”后脚本恢复运转。
* **动态防封熔断器**：
    * **安全阀门**：支持单次清理上限设置（建议每日不超过 100 人），触及上限自动脚本自杀，强行防封。
    * **随机延迟**：可在 1s ~ 8s 范围内动态调节取关间隔，仿真人类行为，绕过风控检测。
* **纯净磨砂 UI**：全屏自适应极简控制面板，支持一键折叠收起。

---

## 🛠 安装与使用

1.  确保浏览器已安装 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 插件。
2.  新建用户脚本，将 `X_Unfollow_Master.js` 的代码完整复制并保存。
3.  打开 𝕏 你的关注列表页面：`https://x.com/[你的用户名]/following`。
4.  等待 2 秒，右侧即会弹出“海王管理大师”控制面板。
5.  调整好参数（建议保持默认的低风控配置），点击 **🚀 启动程序** 即可。
