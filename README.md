# X (Twitter) 批量取消未回关脚本
基于 **Tampermonkey（油猴）** 开发，用于自动取消未回关的 X 账号，支持半自动和全自动两种模式，并提供速度调节、数量限制和运行日志。
---
## 功能
- **半自动模式**
  - 手动滚动页面，脚本自动取消当前可见的未回关账号。
- **全自动模式**
  - 自动滚动页面并取消未回关账号，可后台运行。
- **速度调节**
  - 取消关注间隔：**1～8 秒**
  - 自动滚动间隔：**0～2000 ms**
- **数量限制**
  - 可设置本次最多取消关注人数（默认 **80**）。
  - 达到上限后自动停止。
- **运行日志**
  - 实时显示已取消关注的账号及运行状态。
- **重复检测**
  - 自动记录已处理账号，避免重复操作。
---
## 安装
1. 安装 **Tampermonkey（油猴）**。
2. 新建脚本。
3. 将 `script.js` 内容复制进去并保存。
---
## 使用
1. 打开你的关注列表：

https://x.com/你的用户名/following

2. 等待页面加载完成。
3. 设置参数：
- 本次清理上限
- 取消关注间隔
- 自动滚动速度
4. 选择运行模式：
- **半自动**：手动滚动页面，脚本自动取消。
- **全自动**：脚本自动滚动并取消。
5. 可随时点击停止；达到数量上限后脚本会自动停止。
---
## 参数说明
| 参数 | 范围 | 说明 |
|------|------|------|
| 取消关注间隔 | 1～8 秒 | 每次取消关注后的随机等待时间 |
| 自动滚动间隔 | 0～2000 ms | 全自动模式下的滚动检测间隔 |
| 清理上限 | 自定义 | 达到设定数量后自动停止 |
| 运行日志 | - | 实时显示处理进度和结果 |
---
## 注意事项
- 建议每天取消关注不超过 **100** 个账号。
- 大量账号建议分批、分多天清理。
- 取消关注间隔设置过短，可能增加触发平台风控的风险。
