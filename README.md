# X 批量取消非回关 (海王管理大师版)

一款用于 X (Twitter) 的 Tampermonkey 脚本，帮助用户批量管理关注列表，支持自动化清理非回关账户，并预留了自动回关功能。

---

## 功能说明

### 1. 运行模式

*   **批量清粉（全自动）**：开启“发现目标自动取关”后，脚本会自动向下滚动并执行取关及确认操作。
*   **批量清粉（半自动）**：关闭“发现目标自动取关”后，扫描到非回关账号时脚本会暂停滚动并高亮蓝框锁定目标。用户手动处理或选择继续后，脚本会释放该目标并继续向下扫描。
*   **自动回关**：功能开发中，目前仅做前端 UI 预留。

### 2. 配置参数

*   **取关间隔上限**：单次取关动作后的最大随机等待时间（单位：秒）。
*   **滚屏刷新速度**：页面扫描完当前视窗后，向下滚动的停顿间隔（单位：毫秒）。
*   **本次清理上限**：单次运行累计取关的数量达到该设定值时，脚本将自动停止运行。
*   **发现目标自动取关**：勾选则为全自动流；取消勾选则为半自动人工介入流。

## 安装与使用

1. 确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2. 新建脚本，将编写好的 `javascript` 源码完整复制并保存。
3. 打开 X 网页，进入你的正在关注列表页：
   `https://x.com/[你的用户名]/following` 或 `https://twitter.com/[你的用户名]/following`
4. 页面加载完成后，控制台面板将在 2 秒内自动出现在页面右上角。
