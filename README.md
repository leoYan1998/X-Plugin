# 𝕏 批量取消非回关 (海王管理大师精修UI版)

一款专为 𝕏 (Twitter) 打造的批量清理非回关（单向关注）账户的 Tampermonkey 脚本。

## 核心功能

* **智能识别**：精准识别“关注了你”标签，绝不误伤双向关注的好友。
* **自定义控制**：支持取关间隔上限调节、滚屏刷新速度调节以及单次清理数量上限设置。
* **自动/手动执行切换**：
    * 开启“发现目标自动取关”：自动执行取关并模拟弹窗确认。
    * 关闭“发现目标自动取关”：扫到目标时自动暂停滚动并高亮锁定账户，交由人工处理。
* **低调纯净 UI**：全屏自适应极简磨砂控制面板，支持一键折叠。

## 安装与使用

1. 确保浏览器已安装 **Tampermonkey** 插件。
2. 新建用户脚本，将 `X_Unfollow_Master.js` 的代码完整复制并保存。
3. 打开 𝕏 你的关注列表页面：`https://x.com/[你的用户名]/following`。
4. 控制面板将在页面加载后自动部署在右侧。
5. 配置好参数后，点击 **🚀 启动程序** 即可开始。
